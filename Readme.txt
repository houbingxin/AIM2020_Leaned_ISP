prediction_2020.py is to predict the test images, output image is 448*448*3.
Full_res_predition_2020.py is to predict the full resolution images, output image is 6720*4480*3.

To run the code, just change the input path, model path and output path:
1. Input path: input_path
2. Model path: mdl_path
3. Output path: output_path

1. Python 3.7
2. Tensowflow Keras 

Packages:
1. imageio
2. skimage
3. numpy
4. cv2
5. PIL
